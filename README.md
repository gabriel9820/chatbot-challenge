## :wrench: Running the Project

<br />

- Clone the repository on your machine;

```bash
# Clone
$ git clone git@github.com:gabriel9820/chatbot-challenge.git

# Go to api folder
$ cd ./api

# Run with Docker
$ docker-compose up -d

# Run without Docker
$ dotnet run
```
